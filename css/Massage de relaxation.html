Comment faire le massage?

Le massage relaxant est indique pour les personnes fatigues, pour les personnes qui travaillent beaucoup, pour ceux qui veullent se faire gater.

Le massage se compose d'une suite des mouvements appelles manevres. Ce mouvements sont clasiffies par manevres principales et secundaires.

  1. Les manevres principales:
    1. l'effluerage,
    2. Le petrissage,
    3. Les frictions,
